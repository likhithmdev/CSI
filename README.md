# NonContact-Patient-Monitor

This project implements a non-contact patient monitoring system using ESP32 devices for CSI (Channel State Information) transmission and reception, a Python backend for processing, and a Node-RED dashboard for visualization.

## Project Structure

- **backend/**: Python processing layer including UDP receiver, signal processing, and MQTT integration.
- **dashboard/**: Node-RED related assets, including flows and notes.
- **esp32/**: ESP32 firmware for transmitter, receiver, and nurse station alert system.
- **docs/**: Documentation including architecture, viva answers, and calibration methods.
